# Workshop 3 – SOLID Principles Applied

This folder contains the deliverables for Workshop 3 of the *Domotic Circuit Simulator* project.  
This stage focuses on improving the software architecture through the application of the SOLID principles.

## Contents
- **SOLID Analysis:** Explanation of each principle and how it applies to the domotic simulator.
- **Updated UML and CRC Cards:** Diagrams and class responsibilities refined using SOLID.
- **Python SOLID Snippets:** Short examples demonstrating SRP, OCP, LSP, ISP, and DIP.
- **Reflection:** A half-page analysis describing challenges and insights while applying SOLID.

## Purpose
This workshop strengthens the system’s modularity, flexibility, and maintainability by enforcing industry-standard design principles.
